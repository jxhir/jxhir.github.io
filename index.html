Web Servers: History, Technologies, and Future Directions

IT 3203 Research Project - Milestone #1

Abstract & Introduction

The World Wide Web has fundamentally transformed global commerce, digital services, and human communication. While web browsers command the majority of public attention as user-facing interfaces, web servers act as the hidden engines orchestrating internet operations behind the scenes. Every modern web application, static site, and cloud service relies entirely on web servers to intercept, process, and fulfill incoming client requests.

This project explores web servers as a foundational pillar of modern infrastructure. It analyzes their core client-server architecture, tracks their historical evolution from early static daemons to event-driven powerhouses, maps the current market leaders, and evaluates upcoming paradigms like HTTP/3 and AI-driven server automation.
